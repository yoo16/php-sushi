# Frontend Tasks

## Phase 1: Setup

- [ ] `frontend/` を作成する
- [ ] `Vite + React` をセットアップする
- [ ] `npm scripts` を定義する
- [ ] `js/dist/` への build 出力を設定する

## Phase 2: PHP Bridge

- [ ] `menu.php` に React mount 用の要素を追加する
- [ ] `visit_id` と席情報の受け渡し方法を決める
- [ ] 開発時と本番時のアセット読込方針を実装する

## Phase 3: React UI

- [ ] カテゴリタブを実装する
- [ ] 商品一覧を実装する
- [ ] 注文モーダルを実装する
- [ ] 注文一覧と合計表示を実装する
- [ ] 会計確定フローを実装する

## Phase 4: Cleanup

- [ ] `js/app.js` の置き換え範囲を整理する
- [ ] 文字化けしている表示文言を整理する
- [ ] 使わなくなったフロントコードを削除または退避する

## Phase 5: Verification

- [ ] `npm run dev` で画面確認する
- [ ] `npm run build` が成功する
- [ ] PHP から build 済みアセットを読み込める
- [ ] 注文から会計完了まで通し確認する
